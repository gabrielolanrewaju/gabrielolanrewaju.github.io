Python Project

I conducted Exploratory Data Analysis (EDA) on a dataset comprising 113,937 rows and 81 features related to loan records using python libraries. The dataset encompasses a diverse range of loan variables, including loan amount, loan status, term, prosper score, monthly loan payment, stated monthly income, income range, and many others. The primary goal was to gain insights into the dataset and identify the key features that contribute to predicting loan amounts for individual borrowers.
